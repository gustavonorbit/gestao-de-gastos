# Política de Privacidade – EnsinaJá

**Última atualização:** 05/02/2026

O **EnsinaJá** respeita a sua privacidade e tem o compromisso de proteger os dados dos usuários. Esta Política de Privacidade explica de forma simples e transparente como as informações são tratadas no aplicativo.

---

## 1. Sobre o EnsinaJá

O EnsinaJá é um **diário digital para professores**, criado para auxiliar no registro de aulas, presença, conteúdos, notas e observações, permitindo posterior exportação em planilha.

O aplicativo funciona de forma **offline**, priorizando simplicidade e controle total do usuário sobre seus dados.

---

## 2. Dados coletados pelo aplicativo

O EnsinaJá **não coleta dados pessoais identificáveis** como nome, e-mail, telefone ou documentos.

Os dados inseridos no aplicativo são:

* Informações de turmas criadas pelo usuário
* Registros de aulas
* Presenças e faltas
* Conteúdos, notas e observações

Esses dados são **fornecidos voluntariamente pelo usuário** para uso exclusivo dentro do app.

---

## 3. Armazenamento dos dados

* Todos os dados são armazenados **localmente no dispositivo do usuário**
* O EnsinaJá **não envia dados para servidores externos**
* Não há sincronização em nuvem nesta versão

O usuário mantém controle total sobre as informações registradas.

---

## 4. Compartilhamento de dados

O EnsinaJá **não compartilha dados com terceiros**.

Os dados só saem do aplicativo quando o próprio usuário opta por:

* Exportar informações em formato de planilha
* Compartilhar arquivos gerados manualmente

---

## 5. Anúncios

O EnsinaJá pode exibir anúncios em algumas telas.

* Os anúncios **não têm acesso aos dados inseridos no aplicativo**
* Nenhuma informação de turmas, alunos ou aulas é compartilhada com anunciantes

---

## 6. Permissões utilizadas

O aplicativo pode solicitar permissões básicas necessárias para seu funcionamento, como:

* Acesso a arquivos (para exportação de planilhas)

Nenhuma permissão é utilizada para coleta de dados pessoais.

---

## 7. Exclusão de dados

Como os dados são armazenados localmente:

* O usuário pode apagar todas as informações diretamente pelo aplicativo
* A desinstalação do EnsinaJá remove todos os dados armazenados no dispositivo

---

## 8. Público-alvo

O EnsinaJá é destinado a **usuários adultos**, especialmente professores.

O aplicativo **não é direcionado a crianças**.

---

## 9. Alterações nesta Política

Esta Política de Privacidade pode ser atualizada para refletir melhorias no aplicativo.

Sempre que houver alterações relevantes, a data de atualização será modificada.

---

## 10. Contato

Em caso de dúvidas ou solicitações relacionadas à privacidade, o usuário pode entrar em contato pelo canal disponibilizado dentro do aplicativo.

---

Ao utilizar o EnsinaJá, você concorda com esta Política de Privacidade.
